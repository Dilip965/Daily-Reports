# Daily Report App

A simple daily reporting tool for employees.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Initialize database:
   ```bash
   npx prisma db push
   npx tsx prisma/seed.ts
   ```

3. Run the app:
   ```bash
   npm run dev
   ```
